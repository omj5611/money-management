# Money Management

수입 입력 없이 고정지출, 소비지출, 저축 내역, 저축 목표를 관리하는 모바일 웹앱입니다.

## 실행

별도 설치 없이 `index.html` 파일을 브라우저에서 열면 실행됩니다.

## 구현된 기능

- 고정지출 등록, 수정, 이번 달 완료 체크
- 출금계좌, 이동처, 자동이체/직접 납부 관리
- 소비지출 및 추가 저축 기록
- 출금계좌 관리
- 카테고리 관리
- 저축 목표 관리와 달성률 계산
- 지출/저축/계좌/카테고리별 분석
- 브라우저 `localStorage` 기반 데이터 저장

## 화면

하단 탭으로 고정지출, 소비지출, 지출 분석, 저축 목표, 관리 화면을 전환합니다.
